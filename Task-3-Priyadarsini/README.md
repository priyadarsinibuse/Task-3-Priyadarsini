# TASK-3-PRIYADARSINI

## 📌 Project Description
This is a backend API project built using Node.js, Express.js, and SQLite database.  
It performs CRUD operations (Create, Read, Update, Delete) using REST APIs.

## ⚙️ Technologies Used
- Node.js
- Express.js
- SQLite3


## 🚀 Features
- REST API development
- SQLite database integration
- Create table automatically
- Insert data into database
- Fetch all records
- Update records
- Delete records



## ▶️ How to Run
1. Open terminal in project folder
2. Install dependencies:npm install
3. Start server:node server.js
4. Open browser:http://localhost:3000


## 🔗 API Endpoints
- GET /students → Get all students
- POST /students → Add new student
- PUT /students/:id → Update student data
- DELETE/students/:id → Delete student data
